1 GUI Programming in Java |Madhu T. GUI Programing with Java Java Provides 2 Frameworks for building GUI-based applications. Those are  AWT-(Abstract Window Toolkit)  Swing AWT-(Abstract Window Toolkit): • AWT (Abstract Window Toolkit) is an API to develop GUI or window-based applications in java. • The java.awt package provides classes for AWT API such as TextField, Label, TextArea, Checkbox, Choice, List etc. • AWT components are platform-dependent i.e. components are displayed according to the view of operating system.
